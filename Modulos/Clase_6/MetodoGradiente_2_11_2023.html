M√©todo del gradiente#

El m√©todo de descenso del gradiente es una t√©cnica de optimizaci√≥n iterativa utilizada para encontrar el m√≠nimo local de una funci√≥n diferenciable. Es especialmente √∫til para funciones que tienen muchos m√≠nimos locales y globales. Su popularidad ha crecido en las √∫ltimas d√©cadas debido a su eficacia en el entrenamiento de modelos de aprendizaje autom√°tico, especialmente redes neuronales.

Algoritmo del gradiente descendente#

La idea b√°sica detr√°s del m√©todo es simple: dada una funci√≥n objetivo \(f(x)\), queremos encontrar el valor de \(x\) que minimiza \(f\). Para hacerlo, comenzamos en un punto inicial y seguimos los pasos en la direcci√≥n que decrece el valor de \(f\) m√°s r√°pidamente, es decir, en la direcci√≥n del gradiente negativo.

La formula para el c√°lculo del valor siguiente es

\[ x_{i+1} = x_i + Œ± ‚àáf(x) \]

Ac√° \(Œ±\) es el factor de sensibilidad o Tasa de aprendizaje

Un aspecto crucial del m√©todo es la elecci√≥n de la ‚Äútasa de aprendizaje‚Äù, que determina cu√°n grandes son los pasos que tomamos en cada iteraci√≥n. Una tasa de aprendizaje demasiado grande puede hacer que el m√©todo oscile o incluso diverja, mientras que una tasa demasiado peque√±a puede hacer que el m√©todo converja muy lentamente.

El error del m√©todo de puede calcular de varias formas

\[ er = |\frac{x_{i+1} - x_i}{x_{i+1}}| * 100\% \]
\[ |x_{i+1}-x_i| ‚âà 0 \]
\[ |‚àáf(x_i)| ‚âà 0 \]

Como este m√©todo es divergente se debe agregar una condici√≥n de parada. Si no se alcanza convergencia el m√©todo no va a parar, entonces se debe agregar otro par√°metro, el max_iterations que si no se alcanza convergencia entonces cuando se llegue al m√°ximo de iteraciones se frena el algoritmo.

import sympy as sp

def gradient_descent_univariate(f, variable, ci = 0, learning_rate=0.01, max_iterations=1000, tolerance=1e-6):
    """
    Implementa el m√©todo de descenso del gradiente para minimizar una funci√≥n univariada f.

    Par√°metros:
    - f: funci√≥n univariada a minimizar.
    - variable: variable de f.
    - learning_rate: tasa de aprendizaje (tama√±o del paso).
    - max_iterations: n√∫mero m√°ximo de iteraciones.
    - tolerance: tolerancia para la convergencia.

    Retorna:
    - Lista con los puntos de cada iteraci√≥n.
    """

    # Calcula la derivada de f
    derivative = sp.diff(f, variable)

    # Punto inicial (por simplicidad, usamos ci)
    points = [ci]

    for iteration in range(max_iterations):
        # Calcula el valor de la derivada en el punto actual
        gradient_value = derivative.subs(variable, ci)

        # Actualiza el punto actual
        ci -= learning_rate * gradient_value

        # Guarda el punto actual en la lista de puntos
        points.append(ci)

        # Condiciones de parada
        if abs(gradient_value) < tolerance:
            break

    return points

# Ejemplo

points = gradient_descent_univariate(f, x)

import sympy as sp
import numpy as np
import matplotlib.pyplot as plt
from IPython.display import clear_output
import time

def gradient_descent_univariate(f, variable, ci = 0,learning_rate=0.01, max_iterations=1000, tolerance=1e-6):
    """
    Implementa el m√©todo de descenso del gradiente para minimizar una funci√≥n univariada f.

    Par√°metros:
    - f: funci√≥n univariada a minimizar.
    - variable: variable de f.
    - learning_rate: tasa de aprendizaje (tama√±o del paso).
    - max_iterations: n√∫mero m√°ximo de iteraciones.
    - tolerance: tolerancia para la convergencia.

    Retorna:
    - Lista con los puntos de cada iteraci√≥n.
    """

    # Calcula la derivada de f
    derivative = sp.diff(f, variable)

    # Punto inicial (por simplicidad, usamos ci)
    points = [ci]

    # Convertir f a una funci√≥n num√©rica para graficar
    f_lambda = sp.lambdify(variable, f, "numpy")
    x_vals = np.linspace(float(ci) - 10, float(ci) + 10, 400)
    y_vals = f_lambda(x_vals)

    for iteration in range(max_iterations):
        # Calcula el valor de la derivada en el punto actual
        gradient_value = derivative.subs(variable, ci)

        # Actualiza el punto actual
        ci -= learning_rate * gradient_value

        # Guarda el punto actual en la lista de puntos
        points.append(ci)

        # Grafica la funci√≥n y la trayectoria del punto
        clear_output(wait=True)
        plt.figure(figsize=(10, 6))
        plt.plot(x_vals, y_vals, label=str(f))
        plt.scatter(points, [f_lambda(p) for p in points], color='red')
        plt.title(f"Iteraci√≥n {iteration + 1}")
        plt.legend()
        plt.show()

        # Pausa para visualizar la gr√°fica
        time.sleep(0.1)

        # Condiciones de parada
        if abs(gradient_value) < tolerance:
            break

    return points
